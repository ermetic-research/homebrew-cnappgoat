# Homebrew Formula for CNAPPgoat

This repo contains the [Homebrew](https://brew.sh/) formula for
[CNAPPgoat](https://github.com/ermetic-research/cnappgoat)

## Installation

```sh
brew tap ermetic-research/cnappgoat
brew install cnappgoat
```

## Issues, Feature Requests, etc...

If you have any problems related to the installation of CNAPPgoat via Homebrew,
please [open an issue](https://github.com/ermetic-research/homebrew-cnappgoat/issues/new)

Any issues or feature requests related to CNAPPgoat itself should be filed in the
[CNAPPgoat repo](https://github.com/ermetic-research/cnappgoat)
